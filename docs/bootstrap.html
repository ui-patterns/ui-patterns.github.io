Bootstrap Components

Fedora Bootstrap ships with all the components, layout tools & utilities of Bootstrap 4. Check out the Bootstrap 4 Documentation(https://getbootstrap.com/docs/4.0/getting-started/introduction/) for full details on all the elements available in Bootstrap 4.

Fedora Bootstrap only slightly tweaks the styles of a few of these default elements. Listed below are some of the more common Bootstrap 4 components that we use in Fedora Bootstrap based apps, and how they look with the theme.

Alerts
